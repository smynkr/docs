Axiom Layer integrates with over 60 tools across your IT stack. Each integration uses read-only API access to discover assets, track usage, and surface spend data.

## Connection methods

Integrations connect through one of two methods:

- **OAuth** -- authorize directly through the provider's login flow (for example, Google Workspace)
- **Manual** -- provide API credentials such as tokens, client IDs, or domain URLs

To connect an integration, go to **Settings** in the dashboard and select the provider.

## Sync behavior

After you connect an integration, Axiom Layer runs an initial sync to pull your current data. Ongoing syncs run continuously so that new apps, devices, and licenses appear in your dashboard as they are detected.

<Note>All integrations use read-only access. Axiom Layer never creates, modifies, or deletes data in your connected tools.</Note>

## Browser extension

In addition to API-based integrations, Axiom Layer offers a Chrome browser extension for in-browser SaaS detection. The extension identifies applications your team uses by observing browser activity, without requiring admin-level API access to each tool.

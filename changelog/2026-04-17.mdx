<Update label="2026-04-17" description="Week of April 14" tags={["Feature", "Update", "Fix"]}>

## New features

**Google Workspace and Microsoft 365 sign-in** — You can now sign in with your Google Workspace or Microsoft 365 account. This makes it easier for your team to get started without managing separate credentials.

**Redesigned dashboard** — The main dashboard has a new layout with a spend chart, metric cards with sparklines, and a two-column design that puts the most important information front and center.

**Redesigned pages across the app** — The Apps, People, Spend, Contracts, and Renewals pages have all been refreshed with clearer visuals, better filtering, and improved data presentation. Renewals are now grouped by calendar month so you can plan ahead at a glance.

**Shared navigation bar** — A unified navigation bar now appears across all Axiom sites, making it easier to move between products.

**New landing page hero** — The Layer landing page now features a full-page looping 4K video background for a sharper first impression.

**Scroll-reveal animations** — The integrations, pricing, and FAQ sections now animate into view as you scroll, along with animated counters for key stats.

## Updates

**Improved integrations grid** — Integration tiles now show colored icons, a status ring, and a clear connected state so you can quickly see which tools are linked.

**Better sidebar active state** — The sidebar now highlights the active page with a filled background for easier orientation.

**Atlassian connector improvements** — The Atlassian connector form is now properly aligned, and organization users are paginated for smoother setup in larger teams.

## Bug fixes

- Fixed an issue where the Apps page would redirect to login instead of showing an empty state when no organization was selected.
- Fixed the navigation bar overlapping with the marketing nav on certain pages.
- Corrected the Layer brand mark to use the intended hex-network design.

</Update>
